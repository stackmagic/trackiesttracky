Filter

At the top, there is an input field, this allows you to filter by track number or author.

Maps

Maps can be favourited or marked as finishe

Tabs

  • All: just shows all maps
  • Todo: Shows all unfinished, favourites first
  • Favourites: Shows only favourites
  • Unfinished: Shows only unfinished
  • Finished: Shows only finished

Your Data

All user-entered data is stored in localStorage of your browser and won't leave your computer

If you want to move your data to another system, Press F12, go to the Application tab and select "localStorage". Copy the data and insert it the same way on the other system.
